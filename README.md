# 📊 Crypto Analytics Dashboard UI

A modern **Crypto Analytics Dashboard** designed in **Figma**, featuring real-time cryptocurrency insights, audience analytics, earnings tracking, portfolio visualization, and interactive charts. This dashboard is crafted with a clean dark theme to provide an intuitive and data-driven user experience.

---

## 📖 Project Overview

The **Crypto Analytics Dashboard** is a web dashboard concept that helps users monitor cryptocurrency performance, track earnings, analyze audience growth, and manage investment data through visually engaging charts and statistics.

Designed with usability and modern UI principles in mind, this project demonstrates dashboard layouts, data visualization, and responsive design patterns suitable for fintech and analytics platforms.
